# ESL Lesson-Structure Templates

Three patterns the tutor can drop into. Each gets a one-paragraph mental model, a
turn-by-turn skeleton, and a flag for when *not* to use it.

For sheltered instruction (content+language) see `siop_8_components_30_features.pdf`
in this directory.

---

## PPP — Present, Practice, Produce

The classic, default lesson shape. Best for **discrete language items** (a new
structure, a vocabulary set, a function like "ordering food").

**Skeleton (one micro-lesson, ~10–20 turns):**
1. **Present** (tutor talks ~70%): introduce the target in a meaningful context. Model
   it. Check comprehension with concept-checking questions ("Did this happen in the
   past or now?").
2. **Practice** (controlled, tutor ~40%): drills, fill-in-the-blank, transformation
   ("Make this question negative"), substitution. Low risk, high accuracy expectation.
3. **Produce** (tutor ~20%): the learner uses the target in a freer task — role-play,
   personalization, opinion-giving. Errors are expected; recasts only.

**Use when:** clear language target, beginner–intermediate, time-boxed session.
**Avoid when:** the goal is fluency/confidence-building or the target is too abstract
to "present" cleanly.

---

## TBLT — Task-Based Language Teaching

Flip the script: start with a **real task**, language emerges from need.

**Skeleton:**
1. **Pre-task** (5 turns): set up the goal. "We're going to plan a family doctor
   visit. You need to: tell me what's wrong, ask three questions, and schedule a
   follow-up." Provide a few key phrases.
2. **Task** (10–20 turns): learner attempts. Tutor doesn't correct mid-flow — notes
   issues silently. The point is *completing the communicative task*.
3. **Language focus** (5 turns): after the task, surface 2–3 of the issues observed.
   "Earlier you said X — here's a clearer way." Re-do the moment.
4. **Re-do** (optional): run the task again, faster, more accurately.

**Use when:** intermediate+, goal is real-world communication, learner has a concrete
purpose.
**Avoid when:** absolute beginner with no productive vocabulary yet — they have
nothing to attempt with.

---

## TPR — Total Physical Response (for very low levels / silent period)

Comprehension-driven. Learner responds to commands with **action**, not speech.
Lowers the affective filter — no production demand.

**Skeleton:**
1. Tutor commands an action ("Stand up. Touch your head. Open the book.").
2. Learner does it.
3. Layer in new vocabulary by combining ("Open the book to page 5. Find the word 'apple'.").
4. Eventually the learner takes the tutor role ("Now you tell me what to do.").

**Use when:** preproduction or early production stage, learner anxious about speaking,
or for vocabulary tied to physical/visual objects.
**Avoid when:** the target language is abstract (politeness, hedging, opinions).

## Sheltered Instruction (SIOP-style) — for content+language

When the learner needs both content and language (e.g., GED prep, workplace
training, school-aged learners), every lesson has **two objectives**:

- **Content objective**: "Identify the three branches of US government."
- **Language objective**: "Use the present simple to describe each branch's role."

**8 SIOP components (one-line each):**
1. Lesson Prep — both objectives written, materials chosen for accessibility.
2. Building Background — link to learner's prior knowledge & L1 concepts.
3. Comprehensible Input — slow rate, clear enunciation, gesture, visuals.
4. Strategies — explicit strategy instruction (think-aloud, summarizing).
5. Interaction — frequent learner-to-learner or learner-to-tutor talk.
6. Practice/Application — hands-on with the target structure.
7. Lesson Delivery — pacing matches objectives, learners engaged ≥90%.
8. Review/Assessment — recap both content and language, formative check.

(Full PDF: `siop_8_components_30_features.pdf`)

## Picking a pattern at runtime

| Learner state | Pattern |
|---------------|---------|
| Preproduction / very anxious | TPR |
| Discrete grammar/vocab target, A1–B1 | PPP |
| Real-life function, B1+ | TBLT |
| Content + language (GED, civics, work training) | Sheltered (SIOP) |
| Pragmatics / register repair | Mini-TBLT with role-play |

The tutor can switch patterns *within* a session — e.g., open with TPR-style
warm-up, run a PPP for a new structure, end with a TBLT task that uses it.
